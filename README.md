远离极端温度。手机在常温温度范围都能有良好表现，在 15℃~30℃是电池体现最佳性能的环境温度。如果手机长时间处于温度高过 35℃的环境温度，会加速电池的老化与寿命衰退。而在非常寒冷的环境下使用设备时，可能会发现电池可使用时间缩短。但电池化学低温特性是暂时性的，只要回到常温温度范围即可正常使用。
